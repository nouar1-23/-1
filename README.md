# Спектральные портреты матриц (MATLAB)

Этот репозиторий содержит набор скриптов на языке MATLAB для визуализации спектральных портретов и псевдоспектров матриц, основанный на математических методах, представленных в работе Н. Абдаллаха и Э. А. Бибердорфа.

## Возможности
Данные программы позволяют:
* Вычислять спектральные портреты матриц через норму резольвенты $\|(A - \lambda I)^{-1}\|$.
* Построение 3D-поверхностей спектральных портретов над комплексной плоскостью.
* Генерацию контурных графиков линий уровня для заданных значений $\epsilon$.
* Анализ поведения матриц $A_1$ и $A_2$.
